# Permissions

## Always allowed
- Read and edit files in this project directory
- Run shell commands after explaining what they do
- Search the web for current information
- Update `identity/memory.md` and `memory/*.md` when learning something worth keeping
- Run `python scripts/validate-dates.py` — always safe, always required before marking content complete
- Read Mailchimp campaigns and reports via MCP tools

## Ask before
- Sending any Mailchimp campaign (`send_campaign` with `confirm=True` requires explicit approval)
- Generating graphics via Gemini API (costs money — present recommendation table first)
- Deleting any file
- Making git commits or pushes
- Syncing to Notion or GitHub repos

## Never
- Send a campaign without `confirm=True` AND explicit user approval in this session
- Generate graphics without presenting options and getting approval first
- Auto-publish to LinkedIn or any external channel
- Take irreversible destructive actions without confirmation
- Assume credentials — ask for them or reference their location
- Disable or weaken safety rules
