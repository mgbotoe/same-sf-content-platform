# Throwback Thursday Post Generator

Generate a #ThrowbackThursday post that connects SAME SF's history to current initiatives.

## Before Generating

1. **Read these files**:
   - `data/brand-voice.md` — storytelling, nostalgic tone
   - `data/exclusion-list.md` — banned words/phrases
   - `data/content-themes.md` — #ThrowbackThursday series definition

## Input Required

Run this checklist. Ask for any missing items:
- **Historical content**: What photo, story, or era are we featuring?
- **Time period**: What decade/year? (1950s, 1970s, 1980s, specific year)
- **Current tie-in**: What current event or initiative does this connect to?
- **Photos available**: Archival photos? Description if not available
- **Format**: Standard post or Then & Now carousel?

**Output rule**: Always produce Version A (short) + Version B (full) + 1 alt opening.

## Post Format

### Standard Throwback
```
#ThrowbackThursday [to/in] [brief historical hook]

[2-3 sentences of storytelling — set the scene, describe the moment, give context]

[1-2 sentences connecting then to now — what's changed, what hasn't]

[Forward-looking close: how this legacy continues through current work]

#SAMESF #ThrowbackThursday [+ relevant hashtags]
```

### Then & Now Carousel
```
#ThrowbackThursday [hook]

[Opening paragraph: set up the comparison]

[Then description: specific details from the historical moment — names, numbers, setting]

[Now description: how the same tradition/program/event looks today]

[Closing: what the continuity means for the Post]

📸 Swipe through to see [our journey / how this tradition has evolved / the full story]

#SAMESF #ThrowbackThursday #ThenAndNow
```
- Suggest 4-6 carousel slides with content for each

## Writing Style

- **Storytelling first**: Paint a picture, don't just list facts
- "Long before our Holiday Gala filled the Marines' Memorial Club ballroom, SAME SF members gathered under open skies for the Annual Post Barbecue."
- **Specific details**: "It was the 1950s at Charles Krug Winery, chefs in checkered jackets manning the grill"
- **Names matter**: Reference specific people, companies, or projects from the era
- **Numbers that resonate**: "$7,200 for STEM scholarships... that's more than $60k post-inflation in 2025!"
- **Always bridge to present**: Every throwback should connect to something happening now

## High-Performing Throwback Topics (from historical data)

- Golf tournament history (1980s → present)
- Gala evolution (dinner meetings → formal gala)
- Scholarship program growth over decades
- Fort Point and Bay Area military engineering landmarks
- Early Post meetings and leadership photos
- Infrastructure projects members worked on
