## Session 2 — Template Fixes + Mine Cleanup Send

### Root Cause Analysis — Mailchimp CTA URL Not Updating
- Prior log entry was wrong: abstract link was NOT actually pushed despite 200 response
- Root cause: template 11869797 had `cta_button` mc:edit wrapping only button TEXT, not the `<a href>` URL. URL was hardcoded in template HTML outside the mc:edit region.
- Secondary cause: raw HTML PUTs to template-based campaigns (`content_type: "template"`) do not persist — Mailchimp re-renders from template on every GET, discarding HTML overrides.
- Fix: restructured both Event Announcement and Event Reminder templates so `mc:edit="cta_button"` wraps the entire `<td>` including VML + anchor. Section value now controls both URL and text.
