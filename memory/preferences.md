# Content Preferences & Recurring Patterns

## Version Selection
- Version A (short): solemn observances, reminders, TBTs, last calls
- Version B (full): first event announcements, milestones, branch birthdays
- Default: B for first post of week, A for second

## Post Timing
- **7:00 AM Pacific, every post** — consistency = 6.1% vs 4.2% engagement
- Best days: Tue/Wed/Thu primary, Friday morning secondary, avoid weekends
- 24-hour minimum gap between posts
- Links go in post body (not first comment — Dina forgets to add them)

## Email Timing
- Newsletters: Tuesday morning preferred
- Event Announcements: depends on event date — at least 3 weeks prior
- Event Reminders: 1 week prior
- KBYG: 2-3 days prior

## Content Mix
- 60% value / 25% thought leadership / 15% promotional
- Never post 2 promotional posts back-to-back
- Hard cap: 12 posts/month

## Graphics
- Always present image checkpoint table after drafting — never auto-generate
- Real photos beat generated graphics for: events at real venues, TBTs, recaps, member spotlights
- Generated graphics work for: military observances, branch birthdays, abstract concepts
- File naming: `MM-DD-[slug]-graphic.png`

## Exclusion List
- Read `data/exclusion-list.md` before every content generation — non-negotiable

- Added full "Banned Structural Patterns" section (pompous copula substitutes, "Here's" family, negative parallelism, rhetorical self-questions, filler transitions, teacher mode, performed sincerity, conclusion signals, dismissal formula, superficial -ing tails)

## Hashtags
- Always `#SAMESF` (standardized Feb 2026 — not #SAMESFPost)
- 3-5 max per post
- Contextual: `#AECCommunity`, `#MilitaryEngineers`, `#STEMeducation`, `#BayAreaInfrastructure`

## Emojis
- 0-3 max; zero on solemn observances (Memorial Day, Pearl Harbor, 9/11, Veterans Day)
- No em dashes anywhere — LinkedIn posts, emails, all content

## Observance Post Rules (added 2026-05-03)
- Founding/educational facts go in ALL observance posts, including Version A (short)
- No SAME SF logo on observance graphics — the occasion is the focus, not the org
- Closing question rules: solemn observances = tribute line only (no question); celebratory/professional observances = end with a question
